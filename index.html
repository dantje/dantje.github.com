A taste of Cinnamon

Nov 4th, 2012

After toying around with gnome 3 for two ubuntu releases to avoid unity, I finally decided that it just wasn’t what I wanted from an desktop environment. I tried several other options, and finally settled on Cinnamon.

Technically, I’m still running gnome 3 but using Cinnamon it doesn’t feel like a such radicial depature from my old gnome 2 setup, that I grew to like over the years.

There is a nice ppa for Cinnamon, so it can easily be added to a stock ubuntu install:

I observed two small papercuts with the version I’m currently running (1.6.4-0ubuntu1~quantal1):

  • When the desktops effects were still enabled (Menu/Cinnamon Settings/Effects) chromiums’s window sometimes hadn’t the correct size upon first start.

  • Also there is a small, but noticable and irritating, delay when opening the menu.
